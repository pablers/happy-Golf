# Golf Master: Swing Metronome & Scorecard

**Golf Master** es una aplicacin web full-stack diseñada para golfistas, combinando un metrnomo de swing y una tarjeta de puntuacin inteligente.

## Caractersticas Principales
- **Metrnomo de Swing**: Ajusta tu tempo y ritmo de swing con señales auditivas y visuales.
- **Tarjeta de Puntuacin Inteligente**: Registra tus golpes, putts y comentarios hoyo a hoyo.
- **Anǭlisis Visual**: Grǭficos y heatmaps para entender tus tendencias y ǭreas de mejora.
- **Asistente IA (Gemini)**: Cuestionarios dinǭmicos para analizar tu estado fsico y mental antes y despus de jugar.

## Tecnologas
- **Frontend**: React, Vite, Tailwind CSS (en proceso), Context API.
- **Backend**: NestJS, TypeScript, JWT Auth.
- **Persistencia**: Prisma ORM, PostgreSQL.
